Welcome to image processing api

with this app you can use one of our images and apply any size to it and make it yours

How to use this app?

you can use this this app by:

  • go to the image display main route by copying this route (/api/image) or click on this link here
  • then you can chose witch image you want to display or download from available images here
    • snow
    • porsche
    • bugatti
    • coffee
    • city
    • desk
  • now to resize the image you need to pass two parameters (width, height) you can do this by typing those parameters after the image name parameter
  • EX: http://localhost:3000/api/image?image_name=snow&width=200&height=200
